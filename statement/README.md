# 문제 지문

문제 지문의 저작권은 각 문제 작성자에게 있습니다.

문제지 표지는 [박수현(shiftpsh)](https://shiftpsh.com/)님께서 만들어 주셨습니다.

### 파일 설명

* `cover-2024scon.pdf`: 문제지 표지
* `language.tex`: 언어 입출력 가이드
* `rule.tex`: 대회 유의 사항 / 간식 알레르기 유발 물질 안내
